# tennis-booking

Автоматический бот для бронирования слотов на корты клуба Daulet через Altegio (`b551098.alteg.io`).

Слоты открываются за 3 суток до начала ровно в 07:00 Asia/Almaty и выкупаются за секунды. Цель сервиса — претендовать на нужный слот в момент открытия и фиксировать успех до закрытия окна.

## Запуск (после Phase 1)

```bash
# подготовка окружения
uv sync   # или: poetry install

# запуск тестов
pytest

# запуск сервиса
python -m tennis_booking run --config config/
```

## Безопасность и этика

- Сервис обслуживает явное расписание одного владельца — не сканирует слоты массово, не перепродаёт.
- Реальные ФИО/телефоны клиентов хранятся в `config/profiles.yaml`, который **обязан** быть в `.gitignore`.
- В dev/test использовать `--dry-run`. Боевой запуск против Altegio — только с явным флагом и пониманием последствий (это реальная попытка занять реальный слот).
